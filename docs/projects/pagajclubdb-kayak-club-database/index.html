PagajClubDB (Kayak Club Database)

Tech stack: Microsoft SQL Server, T-SQL, ER Modeling, SSMS

View project

PagajClubDB is a normalized relational database designed to manage club members, event arrangements, event images, and internal news. I created schema scripts with clear PK and FK relationships, domain checks, and indexes to support realistic usage patterns and data integrity. The project also includes seed data scripts and query scripts that demonstrate business scenarios such as filtering party events in the clubhouse, listing past events by category, and approving pending news. The deliverable is structured for repeatable execution, validation, and clear documentation through ER diagrams and result screenshots.
