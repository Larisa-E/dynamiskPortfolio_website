KaosTek

Tech stack: PHP, MySQL, MariaDB, Bootstrap 5, HTML, Sessions, mysqli

GitHub repository

Highlights

  • Architecture: Built with PHP, MySQL, MariaDB, Bootstrap 5, HTML, Sessions, mysqli and structured for readability and future extension.
  • Challenge: Modeled relational data carefully to preserve integrity and support realistic reporting/query scenarios.
  • Learning: Strengthened architecture decisions and debugging discipline while turning complex requirements into maintainable output.

View project

KaosTek is a server-rendered PHP application for managing and viewing customers, products, orders, and order details in one dashboard. I implemented session-based authentication, password hashing and verification, and protected routes to keep access controlled. The database layer is normalized and complemented by read-optimized views, including order detail and customer order summary reporting, so the UI can present business data clearly without complex page logic. The project emphasizes practical full-stack fundamentals: clean structure (public vs src), reusable DB connection logic, simple health diagnostics, and a fast local development workflow.
