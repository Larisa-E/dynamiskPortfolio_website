GreenChatMobileMaui

Tech stack: .NET MAUI, C#, XAML, SignalR, HTTP APIs

View project

GreenChatMobileMaui is a cross-platform mobile chat project built with .NET MAUI and structured around real-time communication patterns. The app concept includes user registration and login, thread-based discussions, and live chat messaging through SignalR-backed communication. I organized the project into UI pages, models, and service-layer API logic to separate presentation from networking concerns and support future scaling. This work showcases mobile-focused architecture, cross-platform UI development in XAML, and integration planning for modern backend-driven chat experiences.
