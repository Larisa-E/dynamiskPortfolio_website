Dog API Gallery

Tech stack: HTML, CSS, JavaScript, PHP, Fetch API, JSON, Bootstrap

View project

Dog API Gallery is a frontend-focused web app where users can search dog breeds, optionally filter by sub-breed, and browse images in a paginated gallery. I structured the app into modular JavaScript layers (api.js, ui.js, modal.js, app.js) to keep responsibilities clear and maintainable. The project includes robust request handling with timeout and fallback logic, so the app can recover if the PHP proxy endpoint fails by switching to direct public API calls. I also implemented a full-screen modal with download support, loading and error states, and a responsive layout for a smooth experience across devices.
